# Book Recommendation System (ML + FastAPI)

## Overview

This project builds a machine learning-powered book recommendation system that predicts the type of books a user is likely to enjoy based on their historical interactions.

The system evolves from:
- Genre prediction (supervised learning)
- Content-based recommendations
- Collaborative filtering
- Hybrid recommendation system

## Goals

- Demonstrate end-to-end ML workflow
- Build a production-style API using FastAPI
- Show scalable architecture thinking
- Provide a realistic recommendation engine design

## Architecture (MVP)

User Data → Feature Engineering → ML Model → Prediction API

## Tech Stack

- Python
- Pandas / NumPy
- Scikit-learn
- FastAPI
- Docker
